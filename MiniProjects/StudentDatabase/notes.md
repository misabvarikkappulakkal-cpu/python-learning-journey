# Student Database System

A simple Python-based Student Database Management System that performs CRUD (Create, Read, Update, Delete) operations using CSV file storage.

## Features

- Add Student Records
- View All Students
- Search Student by Roll Number
- Update Student Information
- Delete Student Records
- Persistent Storage using CSV

## Technologies Used

- Python 3
- CSV Module
- File Handling

## Project Structure

StudentDatabase/
│
├── studentdatabase.py
├── students.csv
├── notes.md
└── requirements.txt
