Build a simple command-line Todo application using Python.

The app allows users to:
- Add tasks
- View tasks
- Remove tasks
- Exit program

This project combines multiple Python basics together.

---

# 🧠 Concepts Used

| Concept | Purpose |
|---|---|
| List | Store tasks |
| while loop | Keep app running |
| if/elif/else | Handle menu choices |
| input() | Take user input |
| append() | Add task |
| remove() | Delete task |
| break | Exit loop |

---

# 📦 Important Code Sections

## Empty List

```python
tasks = []
```

Stores all todo tasks.

---

## Infinite Loop

```python
while True:
```

Keeps application running continuously.

---

## User Input

```python
choice = input("Enter choice: ")
```

Allows user interaction.

---

## Add Task

```python
tasks.append(task)
```

Adds new task to list.

---

## Remove Task

```python
tasks.remove(task)
```

Deletes task from list.

---

## Exit Program

```python
break
```

Stops loop and exits app.
