# CPU & RAM Monitor Dashboard

This project is a simple system monitoring tool built using Python. It displays real-time CPU and memory usage statistics, helping users understand system performance.

## 🎯 Objective

The objective of this project is to:

- Monitor CPU usage in real time
- Track RAM consumption
- Learn system monitoring with Python
- Practice using external libraries

---

## 🛠 Technologies Used

- Python 3
- psutil library
- time module

---

## 📚 Concepts Practiced

- System monitoring
- CPU usage analysis
- Memory management
- Python libraries
- Infinite loops
- Exception handling

---

## ⚙️ How It Works

1. The program retrieves CPU usage percentage.
2. It collects RAM statistics.
3. The information is displayed continuously.
4. The dashboard refreshes every few seconds.
