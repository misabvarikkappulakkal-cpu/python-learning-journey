# File Encryption Tool

A simple Python project that encrypts and decrypts files using the Fernet symmetric encryption algorithm from the Cryptography library.

## Features

- Encrypt text files
- Decrypt encrypted files
- Secure key generation
- Easy to use

## Technologies Used

- Python
- Cryptography Library

## Installation

```bash
pip install -r requirements.txt
```

## Run Encryption

```bash
python encrypt.py
```

## Run Decryption

```bash
python decrypt.py
```
