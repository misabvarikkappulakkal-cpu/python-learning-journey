# Terminal Chat App

A simple terminal-based chat application built with Python.

This project simulates a conversation between a user and a chatbot using predefined responses.

## Features

* Interactive terminal chat
* Multiple chatbot responses
* Exit command support
* Beginner-friendly Python project

## Requirements

* Python 3.x

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python chat_app.py
```

## Example

User: hello

Bot: Hello! How can I help you today?

User: how are you

Bot: I'm doing great. Thanks for asking!

User: exit

Bot: Goodbye!
